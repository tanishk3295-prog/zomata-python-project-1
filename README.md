# Zomato Python Project

This project performs data analysis on Zomato restaurant data using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Dataset
Zomato data.csv

## Features
- Data Cleaning
- Exploratory Data Analysis
- Visualization
- Insights Generation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
